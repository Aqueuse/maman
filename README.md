# Journal à deux voix

<img src="maman.png" alt="maman lumineuse" width="300" align="right" />

Un petit outil minimaliste pour pratiquer un **journal intérieur à deux voix** :

- **Moi** : ma voix du quotidien, brute, sincère, parfois exigeante.  
- **Maman** : ma voix de soin intérieure, douce, patiente, constructive.

L’objectif est de créer un espace simple et chaleureux où ces deux parts peuvent dialoguer, sans dépendre d’un outil externe.

---

## Fonctionnalités (V1)

- Ajout de messages dans les deux voix
- Affichage clair, avec bulles différenciées
- Thème neutre en mode *Moi*
- Thème doux et doré en mode *Maman*
- Image d’ambiance (maman lumineuse) qui apparaît en mode *Maman*
- Sauvegarde automatique dans `localStorage`
- Bouton **Effacer** pour remettre le journal à zéro

## Hébergement

Le projet est pensé pour fonctionner **en GitHub Pages**, sans backend, 100% local.

---

## Objectif du projet

Ce journal n'est pas un outil d'IA.  
C’est une aide pour cultiver **sa propre parentalité intérieure**, en devenant une présence douce, stable et encourageante pour soi-même.
