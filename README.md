# Journal à deux voix

<img src="maman.png" alt="maman lumineuse" width="300"/>

Un petit outil minimaliste pour pratiquer un **journal intérieur à deux voix** :

- **Moi** : vous même.
- **Maman** : Une personne aimante, douce, patiente, positive, constructive.

Alternez simplement entre les deux personnes pour vous entrainer à vous traitez de façon respectueuse et encourageante ❤️

*Ce journal a vocation a remplacer la fonction de coaching relationnel des IA conversationnelles telles ChatGPT*
