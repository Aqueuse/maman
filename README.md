# Journal à deux voix

<img width="1032" height="899" alt="visuel" src="https://github.com/user-attachments/assets/188a3376-c6b4-4802-b03e-e09b8a951f71" />

Un petit outil minimaliste pour pratiquer un **journal intérieur à deux voix** :

- **Moi** : vous même.
- **Maman** : Une personne aimante, douce, patiente, positive, constructive.

Alternez simplement entre les deux personnes pour vous entrainer à vous traiter de façon respectueuse et encourageante ❤️

*Ce journal a vocation a remplacer la fonction de coaching relationnel des IA conversationnelles telles ChatGPT*
